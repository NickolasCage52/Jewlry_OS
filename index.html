Jewelry OS

Единая точка входа для презентации: полноценное приложение на Next.js — основной фокус; статический прототип — для быстрого просмотра без деплоя.

Для заказчика Next.js 15 React19

Главное · веб-приложение

Полная операционная оболочка

Каталог и склад, продажи и CRM, контент и публикации по каналам, закупки и ценообразование, онбординг ролей, регламенты, AI-панели (демо-режим) — в одном интерфейсе.

  • Авторизация по ролям и доступ к разделам
  • Навигация и экраны, готовые к показу процессов
  • Современный стек: Next.js, TypeScript, Tailwind
Открыть веб-приложение Как опубликовать на Vercel

Хостинг: рекомендуется Vercel (бесплатный тариф) — подключите репозиторий, укажите корень проекта web, затем вставьте выданный URL в файл site-config.js в корне репозитория.
